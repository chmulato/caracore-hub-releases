Tecnologia — CaraCore Hub

Stack local e autonomia operacional

Backend

Java 11 e Jakarta EE 10. Aplicação modular (core, persistence, api, web). Servidor de aplicações: Apache Tomcat ou WildFly. API REST (JAX-RS/Jersey) para integrações e front-end JSP para a interface de gestão.

Banco de dados — SQLite

O Hub utiliza apenas SQLite como banco de dados. O arquivo de dados fica no servidor (ex.: data/hub.db), sem necessidade de PostgreSQL ou de rede para o núcleo operacional. Migrações com Flyway. Autonomia local: a operação de pedidos, estoque e posições não depende de conectividade.

Integrações e cache

Integração com marketplaces (Mercado Livre, Shopee, Temu) via API e webhooks. Notificações por WhatsApp e Telegram quando configuradas. Redis é opcional (cache); o sistema funciona sem Redis, apenas com SQLite.
