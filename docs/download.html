Artefatos de deploy e releases

O CaraCore Hub é distribuído como aplicação Java (WAR) para deploy em servidor de aplicações (Tomcat). O banco de dados é SQLite local; não é necessário PostgreSQL.

Quando houver releases publicadas, o artefato e as instruções estarão disponíveis em:

Releases → Latest (GitHub)

O código-fonte e o build do projeto estão no repositório chmulato/caracore-hub. Para deploy customizado, consulte a documentação do projeto e o portal de delivery (matriz).
